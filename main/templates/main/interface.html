Интерфейс будет удобно разработать в приложении QtDesigner. Ссылка на скачивание есть на первоначальной странице “Введение”.
После запуска приложения появится окно с выбором шаблона. Необходимо выбрать Main Window и нажать на кнопку “Create”.
В появившемся окне и будет располагаться весь интерфейс. Из левой панели перетаскиваем нужные объекты. Нам будут нужны поля ввода текста, текстовые поля и кнопки. За отображение текста отвечает Label, он находится в группе Display Widgets. Для ввода текста нужны будут Line Edit, он находится в группе Input Widgets. Кнопок нужно будет два вида: Radio Button и Push Button.
Теперь нужно назначить группы для Radio Button. Для этого нужно выбрать нужные кнопки через ctrl, нажать правую кнопку мыши и выбрать “Assign to button group” -> “New button group”. На правой панели появился новый объект, по надобности его можно переименовать для удобной дальнейшей работы.
Так же нужно расположить дополнительное пустое текстовое поле для вывода конечного результата. Для этого добавляем дополнительный Label и удаляем у него содержимое. Не забывайте добавлять понятные и говорящие id к элементам.

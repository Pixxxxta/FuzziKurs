Нечёткая логика
Введение База правил Функция принадлежности Фаззификация Агрегирование Дефаззификация Интерфейс Тест
В рамках технологической практики был пройден курс по созданию экспертных систем с применением принципов нечёткой логики. Где были изучены такие темы как: написание базы правил для выбранной предметной области, фаззификация получаемых показателей, этапы агрегирования и дефаззификации данных, а также создание пользовательского интерфейса для взаимодействия с экспертной системой. В связи с этим, было принято решение сделать дистанционный курс, который поможет студентам понять данную тему и на примере готовой программы написать собственную экспертную систему.
Мною была написана экспертная система по выявлению тахикардии у пользователя. В данной системе нечёткая логика помогает определять насколько вероятна болезнь. Самым удобным способом реализации оказалась Python разработка с использованием модуля PyQt5.
Вся работа будет разделена на 6 разделов
  • База правил
  • Функция принадлежности
  • Фаззификация
  • Агрегирование
  • Дефаззификация
  • Интерфейс
